# Technical Utility Suite
> **A collection of high-efficiency automation tools and architectural research utilities.**

This repository serves as a showcase for the various automation tools, data conversion utilities, and technical research simulations I have engineered to solve specific operational challenges. It focuses on **Python Automation**, **Data Structural Analysis**, and **System Scripting**.

## Featured Utilities

### 1. Chart-to-Table Converter (Computer Vision & Data Parsing)
- **Problem**: Extracting structured numerical data from semi-structured graphical charts.
- **Solution**: Engineered a Python-based parser that interprets visual chart data and converts it into clean, machine-readable tables.
- **Value**: Massively reduces manual data entry for technical research reports.

### 2. Math Notes Automation (Workflow Optimization)
- **Solution**: A suite of scripts designed to automate the formatting and generation of complex LaTeX and Markdown-based technical documentation.

### 3. OS & Architectural Research
- **Insight**: Simulations of core operating system logic and architectural logic gates, demonstrating a deep fundamental understanding of computer systems.

## Engineering Focus
- **Pure Efficiency**: Every utility is designed to eliminate manual repetition and ensure data accuracy.
- **Modular Scripting**: Built using clean, reusable Python modules that can be integrated into larger production pipelines.
- **Robust Parsing**: Advanced handling of data edges and semi-structured formats.

## Technology Stack
- **Languages**: Python, Shell Scripting.
- **Libraries**: (Generic) CSV, JSON, PIL (Imaging), NumPy.
